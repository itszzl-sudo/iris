命令：npx iris-runtime build
定位
Iris 官方编译打包命令
基于 Rust + WebGPU 自研渲染运行时，无 WebView、无浏览器内核，
对现有 Vue 工程做跨端原生编译打包，零侵入改造。
核心能力
读取现有 Vue 工程
自动识别 Vite/Vue3 项目，复用业务代码、SFC 组件、路由、状态管理，无需新建项目模板。
脱离浏览器环境编译
抛弃 DOM/BOM 浏览器 API，
由 Rust 内核 + WebGPU 硬件加速渲染管线
接管布局、样式、图层、矢量 / 图形渲染。
原生桌面端产物构建
编译打包为平台原生二进制程序：
Windows：.exe 独立程序 / 安装包
macOS：原生 App /dmg
Linux：二进制可执行文件
运行时内置打包
内置 Rust 窗口管理器、系统调用层、权限能力、原生托盘 / 快捷键 / 文件系统桥接，
打包后完全独立运行，不依赖 Node、不依赖浏览器。
静态资源内联 & 产物轻量化
压缩前端资源、合并运行时依赖，
依托 Rust 二进制优势，体积远小于 Electron / 常规 Tauri。
样式与布局适配原生渲染
转译 Vue 样式、弹性 / 流式布局为 WebGPU 渲染指令，
实现 GPU 硬件加速界面渲染，高性能低功耗。
一句话极简描述
npx iris-runtime build
基于 Rust+WebGPU 渲染引擎，将现有 Vue 项目编译构建为纯原生桌面端独立应用，
无 WebView、零项目模板、硬件加速渲染、跨平台原生打包。
顺带补一对照，更好区分：
npx iris-runtime dev：启动本地开发渲染环境，热更新 + WebGPU 实时预览
npx iris-runtime build：生产环境编译、优化、打包、输出正式原生安装包
